# Vehicle Insurance EDA Project

## 📌 Project Overview

This project focuses on performing **Exploratory Data Analysis (EDA)** on a Vehicle Insurance dataset to identify customer behavior, insurance response patterns, vehicle trends, and important business insights.

The project includes:

* Data Cleaning
* Data Preprocessing
* Missing Value Handling
* Outlier Detection & Treatment
* Feature Analysis
* Correlation Analysis
* Data Visualization
* Business Insights

This project was created using Python and popular data analysis libraries in Jupyter Notebook.

---

# 🎯 Objective

The main objective of this project is to:

* Understand the structure of the dataset
* Analyze customer and vehicle information
* Find patterns affecting insurance response
* Visualize trends using graphs and charts
* Generate meaningful business insights from the dataset
